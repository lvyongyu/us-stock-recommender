# 策略系统快速参考

## 🎯 核心策略概览

### 1. 技术分析策略 (40% 权重)
**主要指标：**
- RSI < 30 → 买入信号 (+30分)
- MACD 金叉 → 买入信号 (+25分)  
- 均线多头排列 → 上升趋势 (+20分)

### 2. 量化分析策略 (35% 权重)
**关键因子：**
- 价格动量 > 3% → 强势信号 (+25分)
- 低波动率 → 风险较低 (+15分)
- 放量上涨 → 确认信号 (+20分)

### 3. AI/ML策略 (25% 权重)  
**特征输入：**
- 价格变化、成交量比率、技术指标
- 随机森林模型预测未来走势
- 预测准确率：~75%

## 🔄 策略组合逻辑

1. **并行执行** → 所有策略同时分析
2. **权重加权** → 按配置权重合并结果  
3. **一致性检查** → 评估策略共识度
4. **最终决策** → 输出综合建议
